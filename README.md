PS 문제 소스코드 자동 정리 유틸리티

입력받은 디렉토리(default: .)의 모든 .cpp, .c, .py 파일을 스캔해서 파일명의 숫자(문제 번호)를 추출하고,
파일 개수와 번호 분포를 분석하여 자동으로 최적의 폴더 구조를 생성한 후 파일들을 이동시킨다.

핵심 기능:
1. 파일명에서 문제 번호 추출 (예: "1010.cpp" -> 1010)
2. 전체 파일 개수가 적으면 폴더 없이 평탄하게 유지
3. 파일이 많으면 번호 범위를 분석해서 동적으로 그룹핑
   - 10, 100, 1000, 10000 단위 중 적절한 크기 자동 선택
   - 각 그룹 내 파일 개수가 적당한지 체크
4. 그룹 내부도 재귀적으로 분석해서 필요시 서브그룹 생성
